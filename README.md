# Bs.SolidWorks.Tools

Набор полезных утилит для работы с SolidWorks

 ## Bs.SolidWorks.Tools

Для работы требуется установленный на рабочем месте продукт SolidWorks и .Net Framework 4.8

Реализованные команды:
 ### --ExtractConfigurations

Выполняется поиск файлов *.SLDPRT в текущей папке и во вложенных папках. 

Из найденного файла извлекаются 6 проекций детали плюс изометрический вид. Проекции сохраняются в файл 
формата dwg с тем же именем и расположением, что и исходный файл. 

Если в файле *.SLDPRT несколько конфигураций, для каждой конфигурации проекции извлекаются отдельно. В пространстве модели файла dwg
проекции для каждой конфигурации раскладываются по вертикали.
